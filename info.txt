Siemens Pie - программа для генерации бланка ТУ по файлам конфигурации терминалов Siemens.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Программа представляет собой консольную утилиту, написанную на Python.
Для работы программе требуются два файла: XML и XRio.
Эти файлы необходимо получить из DEX файла конфигурации с помощью DIGSI.
Из DIGSI Manager делаем экспорт устройства, выбираем сначала XML, затем XRio.
Имена файлов лучше оставлять одинаковыми. При этом будут экспортированны файлы со ВТОРИЧНЫМИ
значениями уставок.
Программа (файл sp.exe) принимает на вход в качестве параметров имена XML и XRio файла, можно
указать только один файл, второй будет найден по имени, если он лежит в той же директори  и с
тем же именем.
Также можно указать ключь -p - пока не обращаем внимания! сейчас уставки пересчитываются в первичные.
Можно просто перетащить один из файлов, XML или XRio на значек программы (EXE файла).
Для пересчета уставок нужно указать Ктт и Ктн.
Создан файл уставок.
Уставки пересчитаны и округлены в соотстветсии с требуемой точностью.
Необходимо обращать внимание на знаки ? - это должны быть разного рода спецсимволы, вроде альфа или 
фи, но при эскпорте в Xrio файл они теряются, это проблема DIGSI и SiemensPie ничего не может с
этим сделать. Необходимо корректировать вручную.
